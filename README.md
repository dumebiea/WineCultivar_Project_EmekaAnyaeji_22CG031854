# Wine Cultivar Prediction System

## Overview
A Flask-based Machine Learning web application that predicts wine cultivars based on chemical properties.

## Features
- Algorithm: Random Forest Classifier
- Input Features: Alcohol, Flavanoids, Color Intensity, Hue, Proline, Magnesium.

## How to Run
1. `pip install -r requirements.txt`
2. `python app.py`
3. Visit `http://127.0.0.1:5000`
